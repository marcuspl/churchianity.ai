# Source Library (local snapshots)

This folder holds **local snapshots / excerpts** of external sources that are cited in the manuscript footnotes.

**Conventions**

- Each entry includes a canonical URL and a short excerpt sufficient to verify what we’re claiming.
- These are not full archives of every source; where a document is long, we keep the definitional / most-cited passages.
- If an online page moves, this file preserves the “receipt” for why a citation was used.
- **Sources quality rule**: prefer primary documents (councils/dialogues/official texts) where feasible; prefer peer‑reviewed work or major academic presses for science/history; avoid uncited “pop‑neuro” phrasing (soften if uncited).
